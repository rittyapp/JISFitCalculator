# JIS はめあい公差計算アプリ

iOS向けのJIS B 0401 / ISO 286準拠はめあい公差計算アプリです。

## 機能

- **基準寸法**: 1〜500mm対応
- **公差域クラス**: A〜ZC、a〜zc（全28種類）
- **IT等級**: IT1〜IT18（全18等級）
- **はめあい判定**: すきまばめ / 中間ばめ / しまりばめ
- **視覚的な図解**: 穴と軸の公差域を図示

## インストール方法

1. このリポジトリをクローン
2. `JISFitCalculator.xcodeproj`をXcodeで開く
3. iPhoneまたはシミュレータで実行
